# Results Directory

This folder stores output files generated by the optimisation model.
Files here are **git-ignored** since they are regenerated on each run.

## Expected Outputs

| File | Format | Description |
|---|---|---|
| `optimisation_results.xlsx` | Excel | 8 760-row hourly dispatch, storage SoC, demand, and congestion data |
| `dashboard_results.json` | JSON | Structured data for dashboard: meta, capacities, energy mix, LCOE, CO₂, grid congestion, hourly timeseries |

## JSON Schema (dashboard_results.json)

| Top-level Key | Contents |
|---|---|
| `meta` | Objective, discount rate, currency, grid loss factor, technology lists |
| `capacities` | Installed MW (and MWh for storage) per technology, with resource potential |
| `energy_mix` | Annual generation, capacity factor, per-technology LCOE, CO₂, curtailment |
| `lcoe_summary` | Per-technology LCOE/LCOS with CAPEX/OPEX/fuel breakdown, system LCOE |
| `co2_summary` | Per-technology and system CO₂ totals and emission intensity |
| `grid_congestion` | Threshold, congested hours, peak/mean utilisation, monthly breakdown, hourly utilisation array |
| `hourly` | Full 8 760-row array for all generators, storage, and demand |
| `geographic` | (optional) Site name, coordinates, elevation |
